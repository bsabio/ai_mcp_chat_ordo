# Journal Publication Redesign Artifacts

This folder holds browser verification evidence for the journal publication redesign.

## Baseline Routes

1. `/blog`
2. a representative `/blog/[slug]` article route using a published essay or briefing

## Evidence Format

1. Browser screenshots for the relevant route state
2. Short DOM or assertion notes describing what was verified and what remains visually suspect

## Cadence

1. Capture a baseline pass at Sprint 0 completion
2. Capture a fresh pass after each substantive publication-layout change
3. Capture a final pass before Sprint 4 QA sign-off

## Naming Guidance

Use sprint-prefixed artifact names so evidence stays attributable, for example:

1. `sprint-0-blog-index-baseline.png`
2. `sprint-0-blog-article-baseline.md`
3. `sprint-1-cover-composition.png`
